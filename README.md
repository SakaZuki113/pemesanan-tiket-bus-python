## 🚀 Instalasi

Ikuti langkah-langkah berikut untuk menjalankan projek ini di lokal projek:

### 1. Clone Repository
```bash
$ git clone https://github.com/SakaZuki113/pemesanan-tiket-bus-python.git
$ cd pemesanan-tiket-bus-python
```

### 2. Jalankan Projek
```bash
$ python3 main.py
```
